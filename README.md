# PHP From Scratch

A collection of PHP tutorials, exercises, and exam tasks from my university studies and personal learning.

## About

This repository keeps my code while learning PHP, serving as a reference for concepts I've covered and a place to add new material as I continue studying.

## Topics Covered

- PHP fundamentals: variables, data types, arithmetic, functions
- Control structures: conditionals, loops, switches
- Arrays and associative arrays  
- Form handling: radio buttons, checkboxes, input validation
- Web concepts: cookies, sessions, file handling
- Database operations: connections, queries, form integration
- Advanced topics: hashing, null coalescing, server variables
- And more to come as I learn.

## Structure

The code is organized sequentially as I learned each concept, with separate folders for exercises and exam preparation. Each file represents a specific PHP topic or technique.

## Download & Exploration

To explore the code, you can simply browse it here on GitHub.
To run it locally, clone the repository and use a local server environment like XAMPP.

```bash
git clone https://github.com/Gloomcaller/PHP-From-Scratch.git
```

This serves as my personal knowledge base for PHP development and university exam preparation.
